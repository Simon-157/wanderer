API Endpoints

Endpoint Method Description
/api/v1/auth POST Authenticate user
/api/v1/users GET Get all users
/api/v1/users/:id GET Get a user by ID
/api/v1/users POST Create a new user
/api/v1/users/:id PUT Update a user by ID
/api/v1/users/:id DELETE Delete a user by ID
/api/v1/posts GET Get all posts
/api/v1/posts/:id GET Get a post by ID
/api/v1/posts POST Create a new post
/api/v1/posts/:id PUT Update a post by ID
/api/v1/posts/:id DELETE Delete a post by ID
